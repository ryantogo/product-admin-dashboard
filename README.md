# Product Admin Dashboard (Cloud SQL + Google Cloud)

Fullstack app with:
- Backend: Node.js + Express (Cloud Run ready)
- Frontend: React + Tailwind (Firebase Hosting ready)
- DB: Google Cloud SQL (MySQL)
